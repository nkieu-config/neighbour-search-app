# Neighbour Country Search App 🌍

A responsive web application that allows users to search for countries and view detailed information including neighboring countries.

## 🌐 Demo

👉 **[Visit here](https://nkieu-neighbour-search.vercel.app/)**

Enter any country name (e.g., "Thailand", "Egypt", "Germany") to see detailed information about the country and its neighbors.

---

## ✨ Features

Core Functionality

- 🔍 **Country Search**: Real-time search for any country by name
- 🏳️ **Visual Display**: Country flags with detailed information cards
- 🗺️ **Neighbor Discovery**: Automatically fetches and displays neighboring countries
- ⚠️ **Error Handling**: User-friendly error messages for invalid searches

---

This project was built to practice and demonstrate:

## 🚀 Getting Started

### 🖥️ Run the Site Locally

1. **Clone the repo**

   ```bash
   git clone https://github.com/nkieu-config/neighbour-search-app.git
   ```

2. **Open `index.html`** in your browser
3. **Search** for any country name
4. **Explore the results and neighbor countries**

### No build process or dependencies required - just open and run!

---

## 🌐 API Reference

This app uses the [REST Countries API](https://restcountries.com/) to fetch country data:

- **Search by name**: `https://restcountries.com/v3.1/name/{country}`
- **Search by code**: `https://restcountries.com/v3.1/alpha/{code}`
